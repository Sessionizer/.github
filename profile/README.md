## Hi!
This place is meant for a hobby project called Sessionizer. It is meant to be a place where it's possible to schedule and plan (game) sessions with friends. 
The idea came to me because it can be quite hard to schedule something with friends/colleagues because of responsibilities etc. 

This is also meant as a learning playground for concepts, frameworks and technology in general that i find interesting, making it possible to hopefully bring into my day work.

## For now it includes:
- Basic auth system using Microsoft identity with custom rbac (alongside the gateway)
- Gateway using [Yarp](https://github.com/dotnet/yarp)
